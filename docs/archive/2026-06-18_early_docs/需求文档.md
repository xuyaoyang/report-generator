# 检测报告生成软件 — 需求文档

## 项目目标
开发一款 Windows 桌面软件，通过 Excel 参数表驱动 Word 报告模板，自动生成检测报告（Word/PDF），支持材质单图片自动匹配和报告归档。

## 用户画像
不懂代码的业务人员，会使用 Excel 和 Word 进行日常办公。

## 功能需求

### FR-01: Excel 参数导入
- 程序提供标准 Excel 模板，用户按模板填写
- 支持导入 .xlsx 文件，自动解析参数

### FR-02: 报告生成
- 读取 Word 模板，将参数自动填充到对应位置
- 支持单次参数（封面信息）、型号参数（合格证）、表格参数（检测数据）三类填充

### FR-03: 界面微调
- 导入 Excel 后可在界面上修改参数
- 修改结果影响最终报告输出

### FR-04: 材质单管理
- 支持上传材质单图片（PNG/JPG）
- 按"规格"和"材质"两个维度打标签
- 生成报告时根据产品型号自动匹配对应图片

### FR-05: 多格式输出
- 输出 Word (.docx) 格式
- 输出 PDF 格式（通过 Word COM 转换）

### FR-06: 自动归档
- 按 `项目名称/报告日期` 目录结构自动归档
- 归档路径可配置

### FR-07: 产品扩展
- 首期支持：隔震支座出厂检验报告
- 后续扩展：预埋件、黏滞阻尼器、屈曲约束支撑
- 新增产品类型只需添加模板+配置，无需修改代码

### FR-08: UI 要求
- 简洁直观的桌面应用
- 主色调：淡蓝色
- 双击 .exe 即可运行

## 非功能需求

- **操作系统**：Windows 10/11
- **性能**：生成一份 25 页报告不超过 30 秒
- **可靠性**：模板填充位置准确率 100%
- **打包**：单个 .exe 文件分发

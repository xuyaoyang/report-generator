# 检测报告生成软件 — 技术方案

## 技术选型

| 层面 | 选型 | 版本 | 理由 |
|------|------|------|------|
| 语言 | Python | 3.11+ | Office 文件处理生态成熟 |
| GUI | PySide6 | 6.x | 现代原生桌面 UI，支持 QSS 主题定制 |
| Word 读取 | python-docx | 1.x | .docx 书签替换、表格填充、图片插入 |
| Word COM | pywin32 | 30x | .doc→.docx 转换、PDF 导出（调用本机 Word） |
| Excel | openpyxl | 3.x | .xlsx 读写 |
| 数据库 | SQLite | 内置 | 图片标签存储，轻量无依赖 |
| 打包 | PyInstaller | 6.x | 单文件 .exe 分发 |

## 核心流程

```
1. 用户导入 Excel → excel_reader 解析为 dict
2. param_panel 显示参数，用户可微调
3. word_engine 打开模板 .docx
4. 按 param_mapping 逐项替换书签内容
5. 填充表格数据（力学性能、外观尺寸）
6. image_manager 匹配材质单图片 → 插入指定位置
7. 保存 .docx → 可选 pdf_converter 转 PDF
8. archiver 按项目名/日期 归档
```

## 模板处理方案

原有模板为 WPS 创建的 .doc 格式。处理流程：
1. 用 Word COM 打开 .doc
2. 另存为 .docx
3. 在 .docx 中需要替换的位置插入书签（Bookmark）
4. 程序通过书签名定位并替换内容

## 产品扩展接口

```python
class BaseProduct(ABC):
    @abstractmethod
    def get_param_mapping(self) -> dict: ...
    @abstractmethod
    def get_table_configs(self) -> list: ...
    @abstractmethod
    def validate_data(self, data: dict) -> list[str]: ...
```

新增产品：在 products/ 下新建目录，放入 template.docx + config.json + excel_template.xlsx。

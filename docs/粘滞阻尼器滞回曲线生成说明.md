## 报告只需要的曲线

只需要生成 Excel 样例中 `试验曲线表` 对应的第一张曲线图：

- 标题：`试验力-位移曲线`
- 在样例 `Result` sheet 中的位置：第 84 行第 1 列
- 样例图片尺寸：`550 x 350 px`
- 不需要生成：
  - `试验力-位移曲线（第1次）`
  - `试验力-位移曲线（第2次）`
  - `试验力-位移曲线（第3次）`
  - `控制曲线`

## dcv 原始数据格式

`.dcv` 文件是 GB18030 编码的文本文件。

文件前半部分是键值参数，例如：

- `SampleNumber`
- `DampingCoefficient`
- `SpeedIndex`
- `f1`
- `DesignedFm`
- `uo`
- `LimitDisplacement`
- `Frequency`
- `Amplitude`

从以下表头开始是制表符分隔的采样数据：

```text
Order	Load(kN)	Dis(mm)	Defor(mm)	Time(s)	HalfCycle	Defor1(mm)	Defor2(mm)
```

其中后续生成滞回曲线主要使用：

- X 轴：`Dis(mm)`
- Y 轴：`Load(kN)`
- 分段辅助：`HalfCycle`

## 样例规律

样例 Excel 中第 6-10 行的循环统计与 `.dcv` 中 `HalfCycle` 分段对应：

- 第 1 次循环：`HalfCycle=1` 和 `HalfCycle=2`
- 第 2 次循环：`HalfCycle=3` 和 `HalfCycle=4`
- 第 3 次循环：`HalfCycle=5` 和 `HalfCycle=6`
- 第 4 次循环：`HalfCycle=7` 和 `HalfCycle=8`
- 第 5 次循环：`HalfCycle=9` 和 `HalfCycle=10`

第一张“试验力-位移曲线”是总体滞回曲线图，不需要拆成单次循环图。

## 当前实现方式

程序不会要求用户提供 `.dcv` 原始数据。

当项目基本信息中的 `是否需要滞回曲线` 选择 `是` 时：

1. 程序读取力学性能数据中的 `最大阻尼力(kN)`、`设计位移(mm)`、`阻尼力指数(α)`。
2. `core/hysteresis_curve.py` 从 `products/viscous_damper/curve_templates.json` 中选择最接近的真实 `.dcv` 曲线模板。
   - 当前模板库包含 14 条真实曲线。
   - 同一份报告内会记录已使用模板，优先为不同型号分配不同模板，避免一眼看出复用同一条曲线。
3. 程序将模板曲线按目标参数缩放：
   - X 轴按 `目标设计位移 / 模板设计位移` 缩放。
   - Y 轴按 `目标最大阻尼力 / 模板实际峰值阻尼力` 缩放。
4. 缩放后保留真实原始曲线的起始段、终止段、工装间隙小台阶、外圈抖动和循环重合度。
5. `core/doc_composer.py` 在处理粘滞阻尼器 `08_mechanical.docx` 时，用生成的 PNG 替换模板中的曲线占位图。

当 `是否需要滞回曲线` 选择 `否` 时，程序使用 `08-1_mechanical.docx`，不插入曲线图。

样例图视觉特征：

- 蓝色曲线
- 绿色虚线网格
- 标题：`阻尼力—位移曲线`
- X 轴：`位移(mm)`
- Y 轴：`阻尼力(kN)`
